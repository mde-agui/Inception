Login

    Default Username: admin

    Default Password: admin

    It will prompt you to change the password.

Add Prometheus Data Source

    Click ⚙️ “Configuration” → “Data Sources”.

    Click “Add data source” → choose Prometheus.

    Set URL to: http://prometheus:9090

    Click “Save & Test”.

Import a Dashboard

    Click “+” → “Import”.

    You can:

        Use a prebuilt dashboard from Grafana.com.

        For example:

            Node Exporter Full (ID: 1860)

            NGINX Metrics (ID: 12708)

            PHP-FPM (ID: 15796)

    Paste the ID, click Load, then assign Prometheus as the data source.
